This Software only works with Google Chrome version  79.0.3945.88 or later.
It also requires that the computer has python 3.8 installed.
This software is dependent on d3 v5 for the visualizations and bootstrap for the overall look and feel, the requirements are the same.
Please consider that this software was only tested on Windows 10 and macOS Catalina.

It is designed for screens with resolution os 1080 by 1920 or higher.
If the dashboard does not appear in its entirety in your display please consider ctrl + '-' on windows or cmd + '-' on mac until you see all the idioms. 
Or 'change the size of text, apps and other items' in the settings to 100%. Access this settings by right clicking in the desktop and choosing 'Display settings'.


In order to run it please go to the directory of this file an run the following code.

If you are using Windows 10 or later:
python -m SimpleHTTPServer 8888

If you are using macOS Catalina or later:
python2 -m SimpleHTTPServer 8888

If non of these works please try:
python -m http.server 8888

Then access the following url:
http://localhost:8888/what-makes-us-healthy.html

The usage of this piece of software does not require internet connection.
